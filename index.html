What is CoffeeKit?

Simply, CoffeeKit lets you write native iOS apps using JavaScript. The language bindings are expressed in terms of CoffeeScript, and the tooling gives some default preferential treatment to CoffeeScript, but JavaScript (or any language that compiles to JavaScript) can be used.

This blog post provides a better introduction to what the code looks like, and there are also a few blog posts about the internals.
