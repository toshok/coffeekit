CoffeeKit 0.1.1 Release Notes

Jul 3rd, 2011

Changes since the last release:

  1. The device bits are now included, so you should be able to build and run your coffeekit programs on device. Note: you should NOT ship things or try to push to the AppStore right now. There are obvious stability reasons(check that version number above), and there’s the fact that JavaScriptCore is a private framework which will get your app blocked.
