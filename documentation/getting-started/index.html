Getting Started

Here’s a simple recipe from going from 0 to CoffeeKit:

1. You need a Mac

We have plans to open up the tooling story to include windows boxes for those of you who love VS, but you’ll always need access to a mac in order to deploy to ios devices (and run in the simulator).

2. You need to be a Apple Developer

You’ll need Xcode4 and the iOS 5 SDK. We don’t run anything older here, and it would be suprising if CoffeeKit worked on older configurations.

3. You need CoffeeKit

Download it by either clicking the “Get it” in the main nav bar above, or clicking here. Installing it will put it in /Developer/CoffeeKit

4. … Profit

From here you can either take a gander at the included samples in /Developer/CoffeeKit/Samples or start a new project with the coffeekit tool. For instructions on how to use it, go here.
